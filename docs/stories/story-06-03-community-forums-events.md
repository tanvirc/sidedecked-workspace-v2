# Story 6.3: Community Forums & Events

## Goal
Launch forums and events hub with moderation workflows.

## Context
Epic 6: Community & Engagement

## Dependencies
- story-06-02-messaging-notifications.md

## Acceptance Criteria
1. Provide forums with topics, threads, replies, and moderation workflows.
2. Enable event creation, RSVPs, reminders, and calendar exports.
3. Allow moderators to flag content, suspend users, and view audit trails for compliance.

## Implementation Tasks
- Build forum data models and UI with moderation queues and approval flows.
- Implement events module with scheduling, RSVP tracking, reminders, and iCal exports.
- Add moderation tools for flagging, suspension, and audit trail review.

## Validation Plan
- Run functional tests covering forum posting, moderation actions, and notifications.
- Validate event lifecycle including creation, RSVP, reminders, and exports via tests.
