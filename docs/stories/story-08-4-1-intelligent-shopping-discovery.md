# Story 8.4.1: Intelligent Shopping Discovery

**Epic**: [epic-08-search-discovery.md](../epics/epic-08-search-discovery.md)
**Status**: not_started
**Domain**: Both

## User Story

_As a buyer, I want intelligent shopping features so that I can find the best deals and discover products that match my needs and budget._

## Acceptance Criteria

- (NOT BUILT) Price comparison across vendors for the same card and condition
- (NOT BUILT) "Best deal" highlighting with savings calculation vs. market average
- (NOT BUILT) Bundle opportunity detection for combined purchase savings
- (NOT BUILT) Condition-adjusted pricing comparison for fair evaluation
- (NOT BUILT) Vendor reliability scoring integration in shopping results
- (NOT BUILT) Personalized shopping recommendations based on deck and collection needs
- (NOT BUILT) Price history display in search results for trend context
- (NOT BUILT) Wishlist integration highlighting wanted cards in all search results
- (NOT BUILT) Budget-filtered shopping results with affordability indicators
- (NOT BUILT) Local seller highlighting for faster delivery preference

## Implementation Notes

Intelligent shopping discovery would overlay market intelligence on all marketplace search results. Price comparison would show condition-adjusted prices from all vendors carrying the same card. Bundle detection would identify opportunities to combine purchases from the same vendor to reduce shipping costs. Vendor reliability scores would surface trustworthy sellers higher in results. Wishlist integration would highlight cards matching the user's active want lists.
